## How It Works

1. **Open the Popup**: Click the extension icon to see all your open windows and tabs.
2. **Manage Tabs**: Drag and drop tabs to move them. Right-click a tab for more options like "Add to Folder" or "Close Tab".
3. **Save to Bookmarks**: Use the "Save", "Sync", or "Override" buttons on any window card to save your session.
4. **Stay Organized**: Use the search bar to find tabs and let the duplicate highlighter show you what you don't need.

### Privacy & Permissions

#### **Single Purpose Description**

```
A comprehensive tab and window manager that allows users to organize open tabs, detect duplicates, and save window sessions to bookmarks for later retrieval.
```

#### **Permission Justifications**

**tabs**

```
Required to list, move, close, and manage the tabs within the extension's popup interface.
```

**windows**

```
Required to list all open windows, create new windows, and move tabs between them.
```

**bookmarks**

```
Required to save groups of tabs to bookmarks and to read existing bookmarks for the "Sync" and "Override" features.
```

**storage**

```
Used to save user preferences, such as the default parent folder for saving bookmarks and popup size.
```

**favicon**

```
Required to display the favicons of open websites next to their titles in the tab list.
```
